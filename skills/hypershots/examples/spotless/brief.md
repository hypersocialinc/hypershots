# Screenshot brief

Filled example — this is the brief behind the set submitted to the App Store with Spotless v1.0 (July 2026). The blank questionnaire comes from `scaffold.sh`; keep the same fields.

- App name / one-line positioning: **Spotless** — check how clean a restaurant really is before you eat there (official health-inspection data, scored 0–10).
- Captures provided (files in assets/):
  - `screen-map.png` — real simulator capture, map with color-graded score pins (1206×2622)
  - `screen-grade.png` — real simulator capture, place page with letter grade + inspection history (1206×2622)
  - `diner.png` — restaurant photo for the hand-built panel-1 place card
  - `tavern.png` — restaurant thumb for the panel-5 chat result card
  - `receipt.png` — app-exported "cleanliness receipt" share artifact (panel 4)
- Panel count + per-panel headline & sub: **5**
  1. shock/hook — "How clean is your\nfavorite restaurant?" / "You might not want to know." (hand-built place card: 3.1 CONCERNING, rats & roaches warning)
  2. map/core value — "Every spot,\ngraded on a map" / "Green is clean. Red means rats." (real map capture)
  3. trust/data source — "Real inspections,\nnot reviews" / "Official letter grades. Pests on record." (real place-page capture, A gradeBadge)
  4. share/delight — "Rate the places\nyou eat most" / "Share your cleanliness receipt." (frameless receipt artifact)
  5. AI bonus — "Ask ChatGPT\nor Claude" / "Spotless answers right in your chat." (hand-built MCP chat screen)
- Stickers / generated assets wanted: die-cut emoji-style cutouts for the shock panel — `sticker-shock.png`, `sticker-sick.png`, `sticker-rat.png`; partner marks `chatgpt-mark.svg`, `claude-mark.svg` (+ `anthropic-mark.svg`, unused alternate) as app-icon stickers on panel 5.
- Device profile(s): iphone-6.9
- Locales: en, es

Arc note: shock → map → trust → share → AI-bonus. The store surfaces ~3 panels before "more", so panels 1–3 carry the install decision — emotional hook first, trust argument by panel 3, delight and differentiators behind the fold.
